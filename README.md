# <div align="center">Centaurus</div>
<hr>
Centaurus is a lore-based addon to Cygnus. As the name suggests, Centaurus adds systems to the Centaurus constellation. For lore reasons, it also changes the Solar System to add more bodies past the four terrestrial planets.
<br><br>
There are five new systems, one of which being Alpha Centauri, a triple star system. The other four are completely original systems, and you will find out more about them as you progress.
